# GrocerAI

## Setup

1. Install dependencies

```bash
npm i
```

2. Start mock API

```bash
npm run api:mock
```

3. Generate native code

```bash
npx expo prebuild --clean
```

## Run

iOS Simulator:

```bash
npm run ios
```

iOS Device:

```bash
npm run ios:device
```

Android:

```bash
npm run android
```
